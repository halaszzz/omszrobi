OMSZ Discord Bot

Telepítés:

1. Telepítsd a Node.js-t:
https://nodejs.org/

2. Nyiss CMD-t ebben a mappában

3. Írd be:
npm install

4. Másold át a .env.example fájlt .env névre

5. Írd bele:
- BOT TOKEN
- CLIENT ID
- GUILD ID

6. config.json-be írd be a channel ID-ket

7. Slash command feltöltés:
node deploy-commands.js

8. Bot indítása:
node index.js
